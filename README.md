## About

A from-scratch recreation of the classic Tetris puzzle game, built in Unity with C#. The goal is a complete, polished Tetris experience with standard mechanics and modern quality-of-life features.

## Planned Features

- Standard Tetris piece set (I, O, T, S, Z, J, L tetrominoes)
- Piece rotation and wall kicks
- Line clearing with combo scoring
- Increasing speed / difficulty over time
- Score tracking and high score persistence
- Next piece preview
- Hold piece functionality
- Game over detection and restart

## Prerequisites

- **Unity** 2021.3 LTS or newer (2D template)
- **Git** for cloning the repository

## Getting Started

```bash
# Clone the repository
git clone https://github.com/astafford8488/tetris.git

# Open in Unity
# 1. Launch Unity Hub
# 2. Click "Open" and select the cloned project folder
# 3. Open the main scene (once added)
# 4. Press Play to run
```
